== Frequently Asked Questions ==

= Does this plugin work with any theme? =

Yes, if that theme use gutenberg editor.

= Does it affect my front-end styles ? =

Absolutely no, Editor Beautifier only affect the block styles in the Back-end area.

= is it a problem if already edited my pages ? =

Not at all, it dose not matter if your pages are already created or you start from scratch, Editor Beautifier will make your editor easier to use.

== Installation ==

1. Upload the plugin to the /wp-content/plugins/ directory.
2. Activate the Editor Beautifier WordPress plugin through the 'Plugins' menu in WordPress.
3. Edit your page or post with Editor Beautifier.
